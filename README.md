Welcome! This repo contains the Interactive Fiction that I have created.

# Staub

My first IF work, a german western written in Inform7. Released in April 2024 for the [Interactive Fiction Grand Prix 2024](https://ifwizz.de/grand-prix-2024.html). Needs Inform7 to be installed on your system.

Base folder: `Staub\Inform7`

Public folder: https://ikeserver.de/Staub 

# PunyStaub

Port of Staub using [PunyInformDE](https://github.com/IkeC/PunyInformDE), my german version of the [PunyInform](https://github.com/johanberntsson/PunyInform) Inform6 library. Released in May 2026. 

Base folder: `Staub\PunyStaub`

Public folder: https://ikeserver.de/Staub/retro.html

# Dust

Inform7 translation of Staub. Released in September 2024 for the [IFComp 2024](https://ifcomp.org/comp/2024).

Base folder: `Dust\Inform7`

Public folder: https://ikeserver.de/Dust

# PunyDust

Port of Dust using the [PunyInform](https://github.com/johanberntsson/PunyInform) Inform6 library. Released in April 2026 with builds for many retro platforms.

Base folder: `Dust\PunyDust`

Public folder: https://ikeserver.de/Dust
