# Staub Retro-Versionen

- Suche den Eintrag passend zu deiner Plattform.
- Ein einzelnes Disk-Image wird direkt verwendet.
- Einen Ordner immer vollstaendig zusammenhalten beim Kopieren oder Entpacken.
- PC-Version: Ordner `PC/` oeffnen und `PLAY.BAT` unter DOSBox oder einem kompatiblen Emulator starten.
- Apple-II-Version: beide Disk-Images zusammenhalten.

## Paketinhalt

| Datei | Plattform | Hinweis |
|---|---|---|
| `Agon/` | Agon Light / Console8 | Ordner (kein Disk-Image) |
| `Apple2/` | Apple II | Nicht getestet |
| `PC/` | DOS / IBM PC | Nicht getestet |
| `staub_amiga.adf` | Amiga | Nicht getestet |
| `staub_atari8bit.atr` | Atari 8-Bit | Nicht getestet |
| `staub_atarist.st` | Atari ST | Nicht getestet |
| `staub_bbc_elk.ssd` | BBC Micro / Electron | Nicht getestet |
| `staub_c128.d71` | Commodore 128 | Nicht getestet |
| `staub_c64.d64` | Commodore 64 | Nicht getestet |
| `staub_plus4.d64` | Commodore Plus/4 | Nicht getestet |
| `staub_cpc_pcw.dsk` | Amstrad CPC / PCW | Nicht getestet |
| `staub_mac.dsk` | Classic Macintosh | Nicht getestet |
| `staub_mega65.d81` | MEGA65 | Nicht getestet |

| `staub_speccy.dsk` | ZX Spectrum +3 | Nicht getestet |

Die Datei `staub.ascii.z5` im Archivstamm enthaelt die ASCII-Version des Spiels
(Umlaute als Digraphen: ae, oe, ue, ss). Im Ordner `Resources` befinden sich
die generierten Grafiken fuer die Lade-Bildschirme.
