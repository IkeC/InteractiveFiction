# Dust Retro Release Package

- Pick the entry that matches your machine.
- If the entry is a single disk image, use that file directly.
- If the entry is a folder, keep the whole folder together when copying or extracting.
- For the PC release, open `PC/` and run `PLAY.BAT` from DOSBox or another compatible PC emulator.
- For the Apple II release, keep both disk images together.
