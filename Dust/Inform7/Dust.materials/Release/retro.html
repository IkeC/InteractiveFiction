Dust was originally released in 2024 as an Inform 7 game in Glulx format. To get Dust working on older platforms, I've ported it to PunyInform so it could be compiled to Z-code (.z5), optimized with ZAbbrev. The retro disk images have then been created using Puny-BuildTools.
