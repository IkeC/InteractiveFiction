Dust was made using Inform 7 and is available for download in Gluxle format (gblorb).

>> Download Game <<

To play the game, you need an interpreter in addition to the game file itself.

  • Lectrote (Windows, MacOS, Linux)
  • Gargoyle (Windows, MacOS, Linux)
  • Spatterlight (MacOS)
  • Frotz (iOS)
  • Fabularium (Android)

With the interpreter, you can play this game and thousands of others Interactive Fiction games.

Dust is also available in other formats for old computers such as C64 and Amiga. Visit the Retro Versions page to see them.
